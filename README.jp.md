# RemoteControl

#### 紹介

このリポジトリは、RDPプロトコルに基づくリモートデスクトップ制御ソフトウェアであり、FreeRDPのソースコードを操作して作成されており、国内のオペレーティングシステムにもともと適合されていないソフトウェアを提供するためです。
