# 数据结构设计
## 一、TCP传输数据包类型

|            数据名称             |    类型    |
| :-----------------------------: | :--------: |
|       RD_IP（远端设备IP）       |   string   |
| RD_Username（远端设备的账户名） |   string   |
|  RD_Password（远端设备的密码）  |   string   |
|     RD_Type（操作指令类型）     |    int     |
|    RD_ImageBit(应用图标数据)    | char[1024] |
|      RD_APP_Name(应用别名)      |   string   |
|  RD_Command_Name（应用程序名）  |   string   |

## 二、TCP传输数据包中操作指令类型

|   操作指令代码   | 数值 |
| :--------------: | :--: |
|   传输应用别名   | 0x01 |
| 传输应用图标数据 | 0x02 |
|  传输应用程序名  | 0x03 |
|     传输结束     | 0x00 |
